**HRmanager is a desktop application for managing employee and applicant records.** While it has a GUI, most interactions are optimized for a CLI (Command Line Interface) workflow.

* If you are interested in using HRmanager, head over to the [_Quick Start_ section of the **User Guide**](UserGuide.html#quick-start).
* If you are interested in developing HRmanager, the [**Developer Guide**](DeveloperGuide.html) is a good place to start.
